# Lwjgl-Menus
A simple template for rendering 2D objects.

This is a basic LWJGL Template using opengl to render 2D Objects in a window.
It has a Minecraft like(Gui)Screen system which allows switching screens.

Supports:
- Squares
- Circles
- Triangles
- Text
- Textures
- Rounded Rects
